Spring-boot JPA CRUD API Application

This application demonstrates a Spring Boot Web-app that performs CRUD operations using Spring Data JPA, on an H2 (In-memory) Database.
This is an Eclipse Project.

Running the app: Options

  1. Execute the class from command-line - org.sameerean.springcrud.boot.Application
  2. From Eclipse, Run As Java Application

URL for end-points

  1. http://localhost:8080/user
  2. http://localhost:8080/user/u12234
